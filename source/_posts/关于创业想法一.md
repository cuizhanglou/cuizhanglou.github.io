下午的时候大哥忽然发消息问关于toB卖鞋的客户来源渠道问题，之前其实也问过，但我也没太在意。这次也很快的回绝了，等我回过味来，忽然发现这里面蕴含着商机。

<!-- more -->

首先，一个完整的销售渠道分为几块，一是技术，二是营销，三是跟单，四是产品及供应链。对于以上四个点，分别刨析如下：
<br/>
1、关于技术，建立一个toB的鞋类网站，其实很简单，不用太多投入和精力，维护成本也几乎为0。
<br/>
2、关于营销，本身我们就有这种渠道，而且一直帮客户做广告培训，很有经验，甚至可以称之为最有经验的外贸投放专员，所以这是我们的优势。
<br/>
3、关于跟单，有朋友是一直做外贸跟单的，在广州深圳附近，只不过前置的渠道被老公司所掌握而已。所以如果前置的客户留言能拿到，后续的跟单销售不是问题。
<br/>

4、产品及渠道，这个需要了解和明确。选品的差异化，定位人群，定位国家，用户画像。
<br/>
所以结合以上信息，其实如果把这条线串起来并不复杂，具体来讲分为以下几个步骤：
<br/>
1、前期调研，寻找同类型的公司和网站，深入分析竞品。查看产品的选择和定位，以及投放盈利用户群体模式，确保思路可行。
<br/>
2、注册公司，然后寻找广告商代开户和充值。
<br/>
3、建立网站，结合竞品明确网站的侧重点并不断强化优化。
<br/>
4、结合用户画像，筛选品类及国家群体，进行广告投放。
<br/>
5、结合广告效果收集客户留言及联系，进行跟单销售。
<br/>
6、结合销售情况进行采购和发货。
<br/>
7、收款及渠道。
<br/>
<br/>
前期可以用熟悉的toB模式做，但是进行到一定规模，可以逐步尝试向toC转，本身就是偏向toC的产品，只不过熟悉toB而已。想一想，心潮澎湃，大有可为。
